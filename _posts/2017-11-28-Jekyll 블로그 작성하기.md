# 1. 로컬 블로그 저장소 이동
```
janghanbyeol-ui-MacBook-Pro:~ j1star$ cd j1star.github.io/
```
<br/>

# 2. 사이트 생성, 변환
```
jekyll serve
```
<br/>

# 3. git 활성화
```
git init
```
<br/>

# 4. 로컬 블로그 저장소 git staging 상태 전환
```
git add .
```
<br/>

# 5. Commit Message 작성
```
git commit -m "write commit message"
```
<br/>

# 6. github 파일 업로드
```
git push -u origin master
```
